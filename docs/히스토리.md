## 결정 기록 (Decision Log)

### 2026-07-17 — 전략 교체·튜닝 및 페이퍼 롤아웃

- **전략 교체:** 보수적(골든+RSI) → **추세추종(골든크로스만, `use_rsi_filter=False`)**. 백테스트에서 보수적이 연 3.3%로 낮아, 위험을 올려 수익을 높이는 방향으로 교체. 두 모드는 `use_rsi_filter`로 전환 가능(옛 전략 보존).
- **파라미터 튜닝:** 12조합 그리드 튜닝 후 **15/60일 + 트레일링 10%** 채택. 최고수익(20/40)이 아닌 상위권 중 MDD 최저(15/60, -23.9%)를 골라 과최적화 위험 최소화. 트레일링 5%→10%가 핵심(추세 유지).
- **레버리지:** 다시 논의됐으나 "현물 백테스터 먼저 완성" 선택 → 보류 유지.
- **페이퍼 롤아웃:** 마일스톤 2 완성 후 **약 1주일 페이퍼 구동(cron 자동)** → 데이터 확인 → 실전(마일스톤 3). cron 매일 KST 00:30 등록.
- **빗썸 API 키:** `.env`에 저장(gitignore). 마일스톤 1·2는 미사용, 실거래 때만 사용. 출금권한 끄기·IP 화이트리스트 권장.
